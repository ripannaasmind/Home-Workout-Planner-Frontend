What is FitHome?

FitHome is a full-stack fitness web application that comes with everything you need to run a professional workout planning platform.

Workout Management

Create, browse, and follow personalized workout plans with difficulty filters, categories, and cover images.

Track weight, body measurements, photos, and workout history with beautiful charts over time.

Earn badges for completing workouts, maintaining streaks, losing weight, and reaching milestones.

Installation

Installation Guide

Follow these steps to get FitHome running on your server in minutes.

1 — Backend Setup

1

Install backend dependencies

cd backend
npm install
2

Create your .env file

Copy .env.example to .env and fill in your values:

PORT=5000
MONGO_URI=mongodb+srv://youruser:yourpass@cluster.mongodb.net/fithome
JWT_SECRET=your_super_secret_key
STRIPE_SECRET_KEY=sk_live_...
STRIPE_WEBHOOK_SECRET=whsec_...
CLIENT_URL=http://localhost:3000
EMAIL_USER=your@email.com
EMAIL_PASS=your_email_password
3

Seed initial data (optional but recommended)

npm run seed            # Seeds all data
npm run seed:features   # Seeds features section
npm run seed:exercises  # Seeds exercise library
npm run seed:products   # Seeds shop products
4

Start the backend server

npm run dev    # Development mode (hot reload)
npm start      # Production mode

Backend runs on http://localhost:5000

2 — Frontend Setup

1

Install frontend dependencies

cd frontend
npm install
2

Create .env.local

NEXT_PUBLIC_API_URL=http://localhost:5000/api
NEXT_PUBLIC_IMGBB_API_KEY=your_imgbb_key
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_live_...
3

Start the frontend

npm run dev    # http://localhost:3000
You're live! Visit http://localhost:3000 for the frontend and http://localhost:5000/api-docs for the Swagger API reference.

FAQ

Frequently Asked Questions

How do I create my first workout plan?
Go to Dashboard → My Workouts and click "Add Workout". Fill in the name, choose a category and difficulty, set the duration, and optionally upload a cover image. Then add exercises from the exercise library.
Can I use FitHome without an internet connection?
FitHome is a web application and requires an internet connection to sync your data, load workouts, and access the shop. However, your progress data is always saved to the cloud so you never lose it.
How do I cancel my subscription?
Go to Dashboard → Billing and click "Cancel Subscription". Your premium access continues until the end of the current billing period. You will not be charged again.
My achievement didn't unlock — what do I do?
Achievements are checked automatically when you complete sessions and log progress. Try refreshing the Achievements page. If it still doesn't unlock, contact support with your username and the achievement name.
Can I upgrade from Monthly to Yearly mid-cycle?
Yes! Go to Dashboard → Billing → Upgrade Plan. You will be charged the prorated difference. Your new billing cycle starts from the upgrade date.
What image formats are supported for uploads?
Supported formats: JPEG, PNG, WebP, GIF. Maximum file size is 5 MB per image. Images are automatically compressed and hosted on our CDN.
How do I access the Swagger API documentation?
With the backend running, navigate to http://localhost:5000/api-docs. This gives you the full interactive API reference powered by Swagger UI.
How do I seed the database with sample data?
Run npm run seed from the backend/ directory. This will populate all collections (exercises, products, features, achievements, etc.) with realistic sample data. Individual seed scripts are also available (e.g., npm run seed:features).
Is payment information stored securely?
Yes. FitHome uses Stripe for all payment processing. Card numbers are never stored on our servers — Stripe handles all sensitive payment data in a PCI-compliant environment.
